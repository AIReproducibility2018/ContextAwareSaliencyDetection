This folder contains the code used in the reproduction attempt. 
Below there is a short description of how to run the two datasets 

**Saliency Detection: A Spectral Residual Approach**
1. Make two folders: Input and Output
2. Download dataset from http://www.houxiaodi.com/assets/papers/cvpr07supp.zip
3. Unzip to this folder (Code)
4. Copy the images in folder "in" to Input folder
5. Run Program.cs
6. Uncomment line 113 in ROC.py
7. Run ROC.py

**learning to predict where humans look**
1. Make three folders: Input, Output and Truth
2. Download images from http://people.csail.mit.edu/tjudd/WherePeopleLook/ALLSTIMULI.zip
3. Unzip to this folder (Code)
4. Download ground-truth from http://people.csail.mit.edu/tjudd/WherePeopleLook/ALLFIXATIONMAPS.zip
5. Unzip to this folder (Code)
6. Run PreProcess1.py
7. Run PreProcess2.py
8. Run Program.cs
9. Uncomment line 114 in ROC.py
10. Run ROC.py
